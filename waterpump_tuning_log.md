# 汽车电子水泵FOC参数调优记录

## 应用场景

- **应用**: 汽车电子水泵
- **功率**: 100-150W
- **电压**: 13.5V（12V汽车系统）
- **转速范围**: 3000-4000 rpm

## 关键约束：低电压系统

### 电压限制分析

在12V汽车系统中，FOC电压约束非常严格：

| 概念 | 计算 | 值 |
|------|------|-----|
| DC母线电压 | Vdc | 13.5 V |
| 线性调制上限 | Vdc/√3 | 7.79 V |
| 过调制上限 | Vdc/2 | 6.75 V |
| 六步换相上限 | 2Vdc/3 | 9.0 V |

**这意味着 ud、uq 的幅值被限制在 ~7.8V 以内（线性调制）**

与之前300V系统仿真对比：
- 300V系统：最大相电压 ≈ 173V
- 13.5V系统：最大相电压 ≈ 7.8V
- 差异：**22倍**

### 对电机参数的影响

低电压电机需要特殊设计：

| 参数 | 高压电机(300V) | 低压电机(13.5V) | 原因 |
|------|---------------|-----------------|------|
| 电阻 Rs | ~1 Ω | ~0.08 Ω | 需要更大电流 |
| 电感 Ld/Lq | ~5 mH | ~0.15 mH | 低感抗，电流响应快 |
| 磁链 KE | ~0.1 Wb | ~0.0045 Wb | 低反电动势，适应低压 |
| 电流 | ~3 A | ~10 A | 相同功率，电流反比电压 |

## 关键发现

### 1. 低电压系统的关键瓶颈：电流能力，而非PI增益

**wp_exp2 最佳配置使用的是最低的PI增益！**
- speed_Kp=0.001（最低）
- speed_Ki=0.5（最低）
- current_Kp=0.5（最低）
- current_Ki=50（最低）

这与高压系统仿真结论完全相反：
- 高压系统：需要提高PI增益来改善响应
- 低压系统：提高PI增益反而变差！

**原因分析**：
- 低电压系统的主要约束是**电压饱和**
- 当uq达到限幅（13V > 7.8V线性调制区），电流环无法输出更多电压
- 此时提高PI增益只会让控制器更激进，但执行能力受限
- 结果：积分器windup，超调，振荡

### 2. 提高电流命令限幅是最关键的调整

从 wp_exp1 → wp_exp2：
- 只改了一个参数：speed_limit 从 5A → 12A
- 结果：稳态误差从 41% → 0.44%

**物理意义**：
- speed_limit 限制的是 iq 命令（速度环输出）
- iq 命令越大 → 电磁转矩越大 → 加速越快
- 在低电压系统中，需要大电流来产生足够的转矩
- 5A电流限幅 → 转矩不足 → 无法达到目标转速

### 3. 电压始终饱和

所有实验中 max_uq = 13V（设定限幅），远超线性调制上限 7.8V：
- 实际进入过调制/六步换相区域
- 这会导致电流谐波增加、控制精度下降
- 但对于水泵应用，这是可接受的（成本优先）

## 电机参数验证

### 理论计算

假设电机参数正确，验证目标转速可行性：

**目标转速 3500 rpm → 电角速度 ωe**
```
ω_mech = 3500 × 2π/60 = 367 rad/s
ω_elec = 367 × 4 = 1467 rad/s (4极对)
```

**反电动势**
```
E = KE × ω_elec = 0.0045 × 1467 = 6.6 V
```

**可用电压**
```
Vdc = 13.5 V
线性调制最大相电压 = 13.5/√3 = 7.79 V
减去反电动势后可用 = 7.79 - 6.6 = 1.19 V
```

**结论**：在3500 rpm时，反电动势已接近线性调制极限，进入过调制是必然的。

### 负载转矩验证

水泵负载特性：T_load = k × ω²

仿真使用 k = 1.7e-6 Nm/(rad/s)²

**3500 rpm时的负载转矩**
```
ω = 367 rad/s
T_load = 1.7e-6 × 367² = 0.23 Nm
```

仿真结果显示稳态转矩 ≈ 0.22-0.23 Nm，匹配良好。

## 实际应用建议

### 1. 电流限制

仿真中 max_iq = 18.4A，需要确认：
- 电机额定电流是否支持
- 逆变器MOSFET是否支持
- 短时峰值电流是否允许（启动阶段）

### 2. 过调制策略

13V电压输出进入六步换相区，建议：
- 实际实现时需要过调制算法
- 或使用六步换相模式在高转速

### 3. 低电感电机的挑战

Ld=0.15mH, Lq=0.18mH 非常低：
- 电流变化率极高：di/dt = u/L ≈ 10V/0.15mH ≈ 67,000 A/s
- 需要**高采样频率**（≥20 kHz）防止电流失控
- 可能需要**电流预测/观测器**

### 4. 测试建议

1. **确认电机参数**：实测 Rs, Ld, Lq, KE
2. **确认电流能力**：电机和逆变器的最大电流规格
3. **调整采样频率**：低电感电机可能需要更高频率
4. **增加保护**：电流限制、过流保护、软启动
